KWIZTIME Unity File Tree (Starter Skeleton)
-----------------------------------------
This zip contains a folder-accurate project skeleton (Assets/*) with placeholder scripts and prefabs.
- Scripts are minimal stubs to help you create the correct structure quickly.
- Prefabs/Scenes/ScriptableObjects are placeholder text files (Unity assets must be created in-editor).

How to use:
1) Create/open your Unity project.
2) Copy the 'Assets' folder contents into your project's Assets/.
3) In Unity, create real Scenes/Prefabs/ScriptableObjects matching these names and replace placeholders.
